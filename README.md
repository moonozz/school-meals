## 💡 프로젝트 설명
- 공공API를 사용하여 제작한 월별 급식 정보 제공 서비스<br/>
- 원스크롤 페이지 및 반응형으로 구현
- Main 영역에 음식 이모지 애니메이션 구현
- 날짜 선택(현재 월 default), 지역 선택 및 학교 검색 > 학교 선택 > 급식 검색 결과 노출 순서로 진행<br />
